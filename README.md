# SABR-model-course-Computing
## SABR Stochastic Volatility Model Group Project Summary
This group project implements and analyzes the SABR (Stochastic Alpha, Beta, Rho) model for option pricing with stochastic volatility. The implementation focuses on addressing the limitations of constant volatility models by treating volatility as a stochastic process. The code is developed in R and includes specialized functions for generating geometric Brownian motion, simulating volatility and price paths, and visualizing the calibrated implied volatility surface.
